### 1. 기존 앱 화면 분석

> `회원가입`

  - 기존 이메일 중복 확인

  - 이메일 인증 번호 발송

  - 이메일 인증 확인

  - 이름 입력

  - 비밀번호 입력

  - 비밀번호 확인

> `로그인`

  - 일반 로그인

    - 이메일 입력

    - 비밀번호 입력

  - 소셜 로그인

    - 카카오 API 활용

    - ID/PW 찾기 기능 미제공

> `ID/PW 찾기`

  - ID의 경우 스프링 이메일 방식 사용

  - PW의 경우 생년월일, 이메일 입력 시 이메일로 PW 전송
